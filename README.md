## About

This repository contains solutions to the [Essential Scala](https://underscore.io/books/essential-scala/) book.

The catalog structure is following:

```bash
essential-scala
├── ...
└── 1-problem-name
    ├── solution.scala
    └── task.md
```

Optionall it can have an alternative solution in
`solution.alt.scala`
